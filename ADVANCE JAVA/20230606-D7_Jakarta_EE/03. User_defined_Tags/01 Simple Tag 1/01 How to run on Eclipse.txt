Observe that web project is ready.
If u get compilation error “The superclass "javax.servlet.http.HttpServlet" was not found on the Java Build Path”,
Open “pom.xml” and add following dependency.

<dependency>
    <groupId>javax.servlet</groupId>
    <artifactId>javax.servlet-api</artifactId>
    <version>3.1.0</version>
    <scope>provided</scope>
</dependency>
