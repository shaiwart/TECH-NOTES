## 1Ô∏è‚É£ Employee class (plain old Java object)

```java
class Employee {
    int id;
    String name;
    double salary;

    Employee(int id, String name, double salary) {
        this.id = id;
        this.name = name;
        this.salary = salary;

    }
```

## 2Ô∏è‚É£ Functional interface (ONE method)

```java
@FunctionalInterface
interface EmployeeCondition {
    boolean test(Employee e);
}
```

This interface represents:

> ‚ÄúA rule that checks an employee and returns true/false‚Äù

## 3Ô∏è‚É£ Create some employees

```java
List<Employee> employees = List.of(
    new Employee(1, "Alice", 50000),
    new Employee(2, "Bob", 70000),
    new Employee(3, "Charlie", 40000)
```

## 4Ô∏è‚É£ Use lambda with the functional interface

### Example 1: salary greater than 50k

```java
EmployeeCondition highSalary =
        e -> e.salary > 50000;
```

That lambda means:

```java
boolean test(Employee e) {
    return e.salary > 50000;
}
```

---

## 5Ô∏è‚É£ Use the lambda

```java
for (Employee e : employees) {
    if (highSalary.test(e)) {
        System.out.println(e);
    }
}
```

### Output:

```
Bob : 70000.0
```

## 6Ô∏è‚É£ Same thing using built-in functional interface (recommended)

Instead of creating your own interface, use `Predicate<Employee>`.

```java

        e -> e.salary > 50000;

    if (highSalary.test(e)) {
        System.out.println(e);
    }
});
```

## 7Ô∏è‚É£ Another lambda example (give bonus)

### Functional interface

```java
@FunctionalInterface
interface BonusCalculator {
    double calculate(Employee e);
}
```

### Lambda

```java
BonusCalculator bonus =
        e -> e.salary * 0.10;
```

### Use it

```java
for (Employee e : employees) {
    System.out.println(
        e.name + " bonus: " + bonus.calculate(e)
    );
}
```

## 8Ô∏è‚É£ Key takeaways (this is the exam + interview gold)

* Lambda = implementation of **one method**
* That method comes from a **functional interface**
* Collections call your lambda for you
* You pass **behavior**, not data
